Detailed Description

Update the 7-segment display with a digit.

Sets the GPIO outputs corresponding to the given digit value. The function does not handle digit multiplexing; it only updates the segment lines (A–G, DP).

Parameters
numberDigit to display (0–9 or predefined segment pattern).
